# Scripts for research

## search_article

Previously on my [Gist](https://gist.github.com/pn11/30b44783eec47ac81a5e), but included in this repository.

### 検索できる雑誌

- PRL
- PRD
- Apj
- PTEP

### 使用例

```sh
./search_article.rb ptep 063C01
./search_article.rb prl 81 1562
```
